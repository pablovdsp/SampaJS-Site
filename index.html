Novidades sobre o JavaScript

JavaScript é uma linguagem muito flexível. Muitos estão acostumados a passar funções como argumentos, ou retorná-las como resultados de chamadas de outras funções. Mas que tal fazer das funções a sua principal abstração de código? Veja nesta palestra como usar o JavaScript em uma abordagem funcional que facilita o fluxo de desenvolvimento e deixa para trás operações baseadas em loops.
